Moov OFAC API Demo

The Office of Foreign Asset Control (OFAC) administers and enforces United States economic and trade sanctions which companies and individuals are required to abide by to disallow transactions with blocked entities. Moov's OFAC service is an HTTP API and Go library to download, parse and serve sanction data for applications and humans. Also supported is an async webhook notification service to initiate processes on remote systems connected with OFAC.
